# 命令列表

C2 下发命令的格式：
AES([uint16: 命令号][字符串: 参数\])

| 命令       | 命令号 | 参数                           | 功能                            |
| ---------- | ------ | ------------------------------ | ------------------------------- |
| echo       | 2      | 单个字符串                     | 木马直接返回该参数的内容        |
| ls         | 3      | 单个字符串，表示路径           | 返回该路径下的文件目录信息      |
| download   | 4      | 单个字符串，表示某一文件的路径 | 返回该路径下的文件              |
| screenshot | 5      | 无                             | 返回当前桌面截图                |
| arp        | 6      | 无                             | 返回宿主机所有网络接口的 arp 表 |
| process    | 7      | 无                             | 返回宿主当前进程列表            |
|            |        |                                |                                 |
|            |        |                                |                                 |
|            |        |                                |                                 |
|            |        |                                |                                 |
|            |        |                                |                                 |
